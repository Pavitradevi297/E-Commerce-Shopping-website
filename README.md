# 🛒 ShopNest - E-Commerce Shopping Website

ShopNest is a responsive and interactive e-commerce shopping website developed using HTML, CSS, and JavaScript. The project provides a modern online shopping experience with product browsing, searching, category filtering, wishlist functionality, cart management, checkout, delivery address collection, payment selection, and order confirmation.

The website is designed with a structured layout consisting of a navigation bar, category sidebar, promotional banner, three-column product grid, shopping features, and footer.

---

## 📌 Project Overview

ShopNest is a frontend e-commerce web application that simulates the essential features of a real-world online shopping platform.

Customers can browse products from different categories, search for products, sort products based on price, add products to their wishlist or shopping cart, manage product quantities, provide delivery information, select a payment method, and place an order.

The shopping cart data is stored using the browser's Local Storage, allowing the selected products to remain available when navigating between the shopping and checkout pages.
